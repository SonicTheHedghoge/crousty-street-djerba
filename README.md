# 🍗 Crousty Street Djerba

> **Le 1er Crousty de Tunisie** 🇹🇳 — God tier restaurant website

## 🔥 Stack

- **Next.js 16** (App Router + Turbopack)
- **Tailwind CSS v4** with custom GTA street-style design system
- **TypeScript**
- **Vercel** deployment ready

## 🎨 Design

GTA Vice City-inspired street food branding:
- 🟣 Electric Blue + Hot Pink neon palette
- 🌑 Ultra-dark background with glow effects
- ✨ Glassmorphism cards with neon shadows
- 🎬 Custom scroll animations & micro-interactions
- 🔤 Bungee + Space Grotesk + Permanent Marker fonts

## 📍 Restaurant Info

- **Location**: Djerba Bourgo Mall, Djerba, Tunisia
- **Cuisine**: Crousty chicken tenders, burgers, wraps, rice bowls, salads, desserts & drinks
- **Social**: [@croustystreetdjerba](https://www.tiktok.com/@croustystreetdjer)

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
