# 🦩 Restaurant Flamant Rose Djerba

> **Gastronomie Méditerranéenne, Poissons Frais & Sunset Lounge** 🇹🇳 — Website pour le Restaurant Flamant Rose à la Zone Touristique de Ghizen, Houmt Souk Djerba.

## 🔥 Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS v4** with luxury coastal glassmorphic design system
- **TypeScript**
- **Lucide React Icons**
- **Framer Motion & Micro-animations**
- **Vercel Deployment Ready**

## 🎨 Features & Highlights

- **Multi-language Support**: Dynamic switching between Français 🇫🇷, English 🇬🇧, and العربية 🇹🇳.
- **Interactive Menu & Currency Converter**: Real-time price display in Tunisian Dinars (TND/DT) and Euros (€), with category filter tabs and dish detail modals.
- **Live Table Reservation Engine**: Multi-step booking modal with instant confirmation tickets and direct WhatsApp/Phone integrations.
- **Google Maps & Social Integration**: Direct links to Facebook page (`https://www.facebook.com/profile.php?id=61553845032970`) and Google Maps coordinates (`33.8723508, 10.9329577`).
- **Visual Photo Gallery**: Full-screen lightbox gallery showcasing fresh catch, clay pot Gargoulette, and sunset terrace vibes.
